# Calculadora de IMC 📊

![Preview do Projeto](./preview.png) <!-- (Adicione uma imagem depois) -->

Projeto funcional de uma calculadora de Índice de Massa Corporal (IMC) com classificação automática, desenvolvido para portfólio.

## 🛠️ Funcionalidades

- Cálculo preciso do IMC (peso/altura²)
- Classificação automática conforme padrões da OMS
- Exibição de resultados com feedback visual (cores)
- Design responsivo (mobile-first)
- Validação de campos

## 🎯 Objetivo do Projeto

- Praticar manipulação do DOM com JavaScript
- Aplicar estilização CSS moderna
- Implementar lógica de programação em um projeto real
- Organizar estrutura de arquivos para projetos web
